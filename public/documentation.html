Welcome to the OP-Api documentation.

The purpose of this API is to compile data on different types of video games and to make available that data for reading, alteration, or outright deleting. The main functions of the server have been implimented already and you can see the list below for instructions on using these functions appropriately.

Get Method

Return all games

Endpoint /games
Will return a list of all games

Return all genres

Endpoint /genres
Will return a list of all genres

Return all studios

Endpoint /studios
Will return a list of all studios

Search for single game

Endpoint /games/"game to search"
Will return the searched for game
ex: /games/Rocket%20League/ will return Rocket Leagues info

Search for users info

Endpoint /users/info
Will return all info on user currently logged in.
Will return error if not logged in.

Put Method

Update user

Endpoint /users/info/
Expected format
{Username: 'Username', --required
Password: 'Password', --required
Email: 'Email', --required
Birthday: Date
make sure you pass the info for username/email & password even if they arent changing.

Post Method

Login

Endpoint /login?Username=USERNAME&Password=PASSWORD
Will request a JWT token for login. Tokens last 30m
replace USERNAME and PASSWORD with your username and password to submit login request

Register User

Endpoint /users
Expected format
{Username: 'Username', --required
Password: 'Password', --required
Email: 'Email', --required
Birthday: Date}

Delete Method

Remove user

Endpoint /users/info
will delete currently logged in user.
Use caution when using method as it will permenantly delete user.

Get Method

Return all users

Endpoint /admin/users
Will return all users info
With great knowledge comes great responsibility

Put Method

Update game

Endpoint /admin/games/"game to edit"/
Expected format
{Name: 'Game Name', --required
Description: 'Game description', --required
Genre : 'Genre name in String format', --required
Studio : 'Studio name in String format', --required
Players: Number} --required
Make sure the genre and studio are registered or error will occur!

Update studio

Endpoint /admin/studios/"studio to edit"/
Expected format
{Name: 'Game Name', --required
Description: 'Game description', --required
Founded: 'String or Date', --required
StillAlive: true or false} --required

Update genres

Endpoint /admin/genres/"genre to edit"/
Expected format
{Name: 'Game Name', --required
Description: 'Game description'} --required

Post Method

Add new game

Endpoint /admin/games
Expected format
{Name: 'Game Name', --required
Description: 'Game description', --required
Genre : 'Genre name in String format', --required --This accepts multiple values but MUST be passed as an array even if theres only one!!
Studio : 'Studio name in String format', --required
Players: Number} --required
Make sure the genre and studio are registered or error will occur!

Add new studio

Endpoint /admin/studios
Expected format
{Name: 'Game Name', --required
Description: 'Game description', --required
Founded: 'String or Date', --required
StillAlive: true or false} --required

Add new genre

Endpoint /admin/genres
Expected format
{Name: 'Game Name', --required
Description: 'Game description'} --required

Delete Method

Remove game

Endpoint /admin/games/"game to remove"
Will permenantly remove game
ex /admin/games/Rocket%20League/ will delete game.

Remove studio

Endpoint /admin/studios/"studio to remove"
Will permenantly remove studio
ex /admin/studios/Respawn%20Entertainment/ will delete studio.

Remove genre

Endpoint /admin/genres/"genre to remove"
Will permenantly remove genre
ex /admin/genre/survival/ will delete genre.

Remove user

Endpoint /admin/users/"user to remove"
Will permentantly remove user
ex /admin/users/user1 will delete user1 use caution as it is not recoverable.
